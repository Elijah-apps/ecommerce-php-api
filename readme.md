# eCommerce API with Leaf PHP By Elijah Ekpen Mensah

This is a simple eCommerce API built with Leaf PHP. The API allows you to manage products, users, and orders.

## Features

- Product Management (CRUD)
- User Registration and Login
- Order Management

## Prerequisites

- PHP 7.4+
- Composer
- MySQL
